# Servarr Config

This repository contains the configuration files and setup for running a media server using Docker on an Ubuntu VM within a Proxmox environment. This setup complements another Docker stack that manages the media server frontend.

---

## Overview
Servarr Config is designed to work with Docker containers, providing infrastructure for: 
- Media organization with tools like Radarr, Sonarr, and Lidarr.
- Automation and simplicity in managing media content.
- Centralized setup to simplify deployment.

---

## Getting Started

### Prerequisites
To use this setup, ensure you have the following installed:
- **Proxmox**
- An Ubuntu VM
- **Docker** and **Docker Compose**
- Access to your actual media frontend
